# Python Programming Course

## Description
Welcome to the **Python Programming Course**! This repository contains all the code examples, exercises, and projects for a comprehensive Python course designed for beginners and intermediate learners from my youtube channel. Whether you're new to programming or looking to strengthen your Python skills, this course will guide you step-by-step through the fundamentals and advanced concepts of Python.

---
## Project Structure
| File | Description |
|-----|-------------|
| 1.hello_world.py | Simple introductory script that prints messages to the console and demonstrates basic Python syntax. |
| 2.Variables.py | Demonstrates how to declare variables and use different data types such as integers, floats, strings, and booleans. |
| 3.list_tuple_set_dictionary.py | Shows how to create and manipulate Python data structures including lists, tuples, sets, and dictionaries. |
| 4.conditional_statements.py | Demonstrates conditional logic using `if`, `elif`, and `else` statements to control program flow. |
| 5.Loops.py | Explains how to use loops (`for` and `while`) to repeat actions and iterate over sequences. |
| 6.Functions.py | Introduces how to define and use functions, pass arguments, and return values. |
| 7.modules_and_packages.py | Demonstrates how to import and use modules and explains the concept of organizing code into packages. |
| 8.Classes.py | Introduces object-oriented programming in Python by defining classes, attributes, and methods. |
| 9.file_handling.py | Demonstrates how to read from and write to files using Python's file handling methods. |
| 10.error_handling.py | Shows how to handle runtime errors using `try`, `except`, `finally`, and how to raise custom exceptions. |
| 11.testing_your_code.py | Demonstrates how to test Python code using the `unittest` framework and assertion methods. |

## Requirements
- Python 3.x
- unittest
